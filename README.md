# 智慧點名系統 (Smart Roll Call System)

這是一個專為部隊/團體設計的網頁版點名管理系統，支援分組管理、多時段公差分配、即時報表統計以及手機版面最佳化。

## 主要功能

### 1. 點名與公差分配
- **分時段管理**: 支援「早點名」、「軍訓」、「晚點名」等多個時段，每個時段的公差狀態獨立儲存。
- **直覺操作**: 人員卡片直接內建公差下拉選單，快速切換狀態 (如: 衛哨, 伙委, 休假等)。
- **雙重篩選**: 支援依據「建置班」與「組別」進行雙重篩選，快速定位未分配人員。

### 2. 進階報表
- **建置班報表**: 依據建置班級統計應到、實到與公差人數。
- **組別報表**: 提供各組別的詳細統計與彙總表格 (Summary Table)。
- **全域彙總**: 複製報表時，自動生成全域統計數據 (總應到、總實到、各類公差總數)。
- **一鍵複製**: 支援將報表內容格式化為文字，方便貼到 LINE 或其他通訊軟體 (自動包含 `應到`、`公差`、`實到` 資訊)。

### 3. 資料管理
- **人員管理**: 支援新增人員 (姓名、建置班、組別)。
- **安全重置**: 「重置所有資料」功能設有三道確認機制，防止誤刪。
- **資料備份**: 支援匯出 JSON 備份檔。

### 4. 介面優化
- **RWD 響應式設計**: 手機版面自動調整，統計卡片自動換行，標題與按鈕自動堆疊，確保操作順暢。
- **視覺化統計**: 清晰的數字卡片顯示應到與實到人數。

## 技術架構
- **前端**: HTML5, CSS3, Vanilla JavaScript (無須編譯，直接運行)。
- **部署**: 支援 GitHub Pages 直接部署。

## 使用說明
1. 開啟網頁。
2. 在「設定管理」新增人員資料。
3. 切換至「點名操作」，選擇時段。
4. 為公差人員選擇對應的狀態。
5. 前往「建置班報表」或「組別報表」查看並複製統計結果。
